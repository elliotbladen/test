# Handover — 2026-05-03

## What was built this session

### 1. Bet ledger infrastructure

**`data/bets/actual_bets_2026.csv`** — extended with 6 new CLV columns:
`model_price, model_line, closing_price, closing_line, clv, source_signal_id`

**`scripts/log_actual_bet.py`** — CLI for the bet ledger:
- `log` — append a new bet, auto-increments bet_id (`2026-0007`, etc.)
- `settle --bet-id 2026-XXXX --result win --return 105.00 --closing-price 2.05` — settles and computes CLV
- `list` — prints ledger with running P/L

**`scripts/bet_performance_report.py`** — auto-generates performance summary:
- Overall P/L, ROI, CLV averages
- Breakdown by sport and market type
- Writes `outputs/bets/performance_summary_2026.csv`

### 2. Post-round tooling

**`scripts/ingest_actuals.py`** — enter actual match results each week:
```bash
# Single game
python scripts/ingest_actuals.py --season 2026 --round 11 \
    --home "Panthers" --away "Storm" --home-score 24 --away-score 18

# Batch
python scripts/ingest_actuals.py --season 2026 --round 11 --from-csv data/import/results_r11.csv

# Check what's missing
python scripts/ingest_actuals.py --season 2026 --round 11 --status
```
Writes to: `results`, `ml_shadow_predictions`, `tier2_performance`.

**`scripts/market_movement_report.py`** — line movement from market_snapshots:
```bash
python scripts/market_movement_report.py --season 2026 --round 11
python scripts/market_movement_report.py --season 2026 --round 11 --market h2h
```
